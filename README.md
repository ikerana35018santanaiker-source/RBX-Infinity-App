# RBX Infinity

RBX Infinity es una aplicación diseñada como plataforma de streaming de películas y series dentro de un entorno interactivo. Su objetivo es ofrecer una experiencia de navegación fluida, moderna y organizada para explorar contenido audiovisual en una interfaz inspirada en servicios de streaming reales.

---

## Características

- Interfaz tipo Netflix con diseño moderno
- Catálogo de películas y series organizado por categorías
- Sistema de búsqueda rápido e intuitivo
- Lista de favoritos personalizada
- Secciones por género (acción, terror, comedia, drama, etc.)
- Animaciones suaves y navegación fluida
- Carga optimizada para mejor rendimiento
- Experiencia de usuario sencilla e intuitiva

---

## Cómo usar RBX Infinity

1. Abre la aplicación RBX Infinity.
2. Explora la página principal con el catálogo disponible.
3. Entra a una película o serie para ver su información.
4. Usa el buscador para encontrar contenido específico.
5. Guarda tus títulos favoritos para acceso rápido.

---

## Funciones principales

- Página principal con recomendaciones
- Sistema de categorías dinámicas
- Vista de detalles de cada película o serie
- Sistema de favoritos
- Navegación entre secciones tipo streaming
- Interfaz adaptada para uso cómodo y rápido

---

## Objetivo del proyecto

RBX Infinity busca recrear la experiencia de una plataforma de streaming moderna, centrada en películas y series, con una interfaz limpia, atractiva y fácil de usar.
